# wz

The Wizzi Factory productions executor

## Installation

`npm install -g wz` 

## Quick Start

  Create the IttfDocuments for a few sample productions (sample1, sample2, sample3)

```bash
$ wz samples
```

  Execute productions

```bash
$ wz sample1
```

```bash
$ wz sample2
```

```bash
$ wz sample3
```
